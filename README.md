# 白酒产业园智慧化建设平台

一个面向白酒企业产业园区的智慧化建设 Demo 平台，涵盖**智慧园区规划**、**物联网感知体系**、**智慧物流运营**和**运维管理**四大核心模块，展示产业园区智慧化、集约化发展的规划设计、项目实施与运营管理最佳实践。

## 功能模块

| 模块 | 描述 |
|------|------|
| 📊 总览看板 | 园区核心 KPI 实时展示、告警信息、物流动态及能耗/生产趋势图表 |
| 📡 物联网感知 | 256 个传感器节点实时监控（温度、湿度、酒精浓度、烟感），数据定期自动刷新 |
| 🚛 智慧物流 | 闸机实时状态、仓库占用率、车辆调度表、近 7 日物流吞吐量趋势 |
| 🗺️ 园区规划 | 园区平面分区图、智慧化建设进度、里程碑时间线 |
| 🔧 运维管理 | 设备台账、维修工单流转、巡检计划、设备健康度看板 |

## 快速开始

本项目为纯前端静态页面，无需安装任何依赖，直接用浏览器打开即可：

```bash
# 方式一：直接打开
open index.html

# 方式二：使用本地 HTTP 服务器（推荐，避免部分浏览器跨域限制）
npx serve .
# 或
python3 -m http.server 8080
```

访问 `http://localhost:8080` 即可查看完整 Demo。

## 技术栈

- **HTML5 / CSS3 / Vanilla JavaScript** — 无需框架，开箱即用
- **Canvas API** — 自绘折线图、饼图、柱状图，无第三方图表库依赖
- 响应式布局，支持桌面端和移动端

## 应用场景

- **规划设计阶段**：通过园区平面图和里程碑时间线直观呈现智慧化建设路径
- **项目实施阶段**：物联网感知体系与设备台账支撑系统落地验收
- **运营管理阶段**：实时看板、告警处理、工单流转保障园区稳定高效运行
