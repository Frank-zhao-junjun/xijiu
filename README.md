# 简易CRM系统

这是一个基于 Next.js 16 + shadcn/ui 构建的简易客户关系管理系统（CRM），适用于PC端和移动端演示使用。

## 功能特性

### 1. 全局搜索 (P0 功能)
- 导航栏全局搜索框
- 支持搜索：客户、联系人、商机、销售线索、报价单、订单、合同、产品
- 实时搜索建议（300ms 防抖）
- 搜索结果分类展示
- 匹配文字高亮
- 键盘导航支持（↑↓ Enter Esc）
- 点击结果跳转详情页

### 2. 仪表盘
- 客户、联系人、销售机会统计
- 销售漏斗可视化
- **销售数据分析**：阶段转化率图表、赢单预测、管道金额分布
- 最近活动记录
- 最近销售机会列表

### 3. 客户管理
- 客户列表（支持搜索和状态筛选）
- 新建/编辑/删除客户
- 客户详情页（查看关联联系人和销售机会）
- 支持PC端表格和移动端卡片两种展示方式

### 4. 销售机会管理
- 机会列表（支持搜索和阶段筛选）
- 完整的销售漏斗阶段（线索 → qualified → 提案 → 谈判 → 成交/失败）
- 新建/编辑/删除销售机会
- 销售漏斗进度可视化

### 5. 联系人管理
- 联系人列表（支持搜索）
- 新建/编辑/删除联系人
- 联系人详情页
- 支持设置主要联系人

### 6. 数据导入导出
- 导入导出管理页面（/data）
- 支持 CSV 和 Excel 格式导出
- 支持 CSV 和 Excel 文件导入
- 数据预览和验证功能
- 详细的错误报告
- 支持数据类型：客户、联系人、销售线索、商机
- 列表页快捷导出按钮

### 7. 报价单管理
- 报价单列表（支持搜索和状态筛选）
- 新建报价单（从商机关联，支持产品明细和折扣计算）
- 报价单详情（编辑、删除、发送、转订单）
- 报价单编辑
- 完整状态流转（草稿 → 已发送 → 已接受/已拒绝/已过期）
- 自动计算小计、折扣、税额和总金额

### 8. 合同管理
- 合同列表（支持搜索和状态筛选）
- 统计卡片（全部/草稿/执行中/已完成/已终止）
- 新建合同（支持从报价单一键创建，自动带入客户和金额）
- 合同详情页（完整合同信息、履约进度跟踪）
- 合同编辑页（仅草稿状态可编辑）
- 履约节点配置（里程碑、预计完成日期）
- 完整状态流转（草稿 → 执行中 → 已完成 → 已终止）
- 履约进度可视化（进度条、节点完成状态）
- 关联管理（客户、商机、报价单）
- 合同条款（标准条款+自定义条款）

### 9. 订单管理
- 订单列表（支持搜索和状态筛选）
- 新建订单（支持从已接受的报价单一键创建）
- 订单详情页（查看订单明细、金额汇总、关联信息）
- 订单编辑（修改产品明细、日期、付款方式等）
- 完整状态流程（草稿 → 已确认 → 待付款 → 已付款 → 已完成 → 已取消）
- 状态进度可视化展示
- 支持订单操作：确认、标记待付款、确认付款、完成、取消
- 自动计算小计、折扣、税额和总金额
- 付款方式管理（银行转账、现金、信用卡等）

### 10. 发票管理
- 发票列表（支持搜索和状态筛选）
- 新建发票（支持从订单创建）
- 发票详情页（含打印预览）
- 发票编辑（仅草稿状态）
- 发票状态管理（草稿→已开票→已收款/逾期/已作废/已退款）
- 自动生成发票号（INV-YYYYMMDD-序号格式）
- 发票打印功能
- 购方税号和开票地址管理

### 11. 活动追踪（P0 功能）
- 完整的操作日志记录（创建、更新、删除）
- 活动时间线展示
- 按实体类型筛选（客户、联系人、商机、合同等）
- 按时间范围筛选
- 按操作类型筛选
- 活动详情查看
- 集成到各实体详情页（客户、商机、合同）
- 分页支持
- 金额自动计算（小计、折扣、税额、价税合计）

### 12. 权限管理 (RBAC)
基于角色的访问控制（Role-Based Access Control）系统

#### 功能模块
- **角色管理**：创建、编辑、删除角色，配置角色权限
- **权限定义**：43 项细粒度权限，覆盖所有业务模块
- **用户角色分配**：为用户分配角色，管理用户权限
- **权限检查中间件**：API 级别和 UI 级别的双重权限控制
- **UI 权限组件**：PermissionGuard、RoleBadge 等组件

#### 预设角色
- **管理员 (admin)**：系统管理员，拥有所有权限
- **销售经理 (sales_manager)**：管理团队和销售流程
- **销售人员 (sales_rep)**：负责客户跟进
- **访客 (guest)**：只有查看权限

#### 权限分类
- 客户权限（查看、创建、编辑、删除、导出）
- 线索权限（查看、创建、编辑、删除、认定）
- 商机权限（查看、创建、编辑、删除、关闭、导出）
- 合同权限（查看、创建、编辑、删除、签署）
- 发票权限（查看、创建、编辑、删除、发送）
- 订单权限（查看、创建、编辑、删除）
- 报价权限（查看、创建、编辑、删除、发送）
- 报表权限（查看、导出、团队报表）
- 设置权限（查看、编辑、自定义字段、工作流、角色管理）
- 用户权限（查看、创建、编辑、删除）

#### 使用示例
```tsx
// 使用 PermissionGuard 组件控制按钮权限
import { PermissionGuard } from '@/components/crm/permission-guard';

<PermissionGuard permission="customers.create">
  <Button>创建客户</Button>
</PermissionGuard>

// 使用 withPermissionGuard 保护 API 路由
import { withPermissionGuard } from '@/lib/api-permission';

export async function POST(request: NextRequest) {
  return withPermissionGuard(request, 'customers.create', async (userId) => {
    // 业务逻辑
    return NextResponse.json({ success: true });
  });
}

// 使用角色标签组件
import { RoleBadge, RoleBadgeGroup } from '@/components/crm/role-badge';

<RoleBadge role="admin" />
<RoleBadgeGroup roles={user.roles} />
```

#### 相关文件
- 权限检查核心：`src/lib/permissions.ts`
- API 权限中间件：`src/lib/api-permission.ts`
- 权限守卫组件：`src/components/crm/permission-guard.tsx`
- 角色标签组件：`src/components/crm/role-badge.tsx`
- 角色管理页面：`src/app/settings/roles/page.tsx`
- 用户角色页面：`src/app/settings/users/page.tsx`
- 数据库 Schema：`src/storage/database/shared/schema.ts`

## 技术栈

- **框架**: Next.js 16 (App Router)
- **UI组件**: shadcn/ui (基于 Radix UI)
- **样式**: Tailwind CSS 4
- **语言**: TypeScript 5
- **状态管理**: React Context + useState
- **数据库**: Supabase

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
# 或者使用 coze dev
```

启动后，在浏览器中打开 http://localhost:5000 查看应用。

### 构建生产版本

```bash
pnpm build
# 或者使用 coze build
```

### 启动生产服务器

```bash
pnpm start
# 或者使用 coze start
```

- **PC端**: 使用侧边栏导航 + 表格展示数据
